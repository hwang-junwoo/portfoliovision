# Frontend

Next.js 14+ 기반 프론트엔드 애플리케이션

## 기술 스택
- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- TanStack Query (React Query)
- Zustand

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## 디렉토리 구조

```
frontend/
├── src/
│   ├── app/              # Next.js App Router
│   ├── components/       # 재사용 가능한 컴포넌트
│   ├── hooks/            # 커스텀 훅
│   ├── lib/              # 유틸리티 함수
│   ├── stores/           # Zustand 스토어
│   └── types/            # TypeScript 타입 정의
├── public/               # 정적 파일
└── package.json
```

*현재 Next.js 프로젝트가 초기화되지 않았습니다. SETUP_GUIDE.md를 참조하여 설정하세요.*
